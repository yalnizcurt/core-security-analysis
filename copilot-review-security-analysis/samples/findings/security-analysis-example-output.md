### 2. [CRITICAL] Code Injection via eval() — CWE-95
**Line:** 73
**OWASP:** A03 — Injection
**Description:** `eval()` is called on user-controlled input (`req.body.expression`). This allows arbitrary code execution on the server.

**Remediation:**
